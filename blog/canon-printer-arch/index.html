Canon MF4770dn Printer Setup on Arch Linux

Eric Semeniuc

September 8, 2019

Install pacaur or substitute for your favourite AUR helper

pacaur -S cndrvcups-lb-bin
sudo system-config-printer
  1. Click “Add” button
  2. Under the “Devices” sidebar, expand the “Network Printer” item
  3. Wait several seconds and “Canon MF4700 (192.168.XXX.XXX)” will appear. Click it
  4. Keep the default host, port number, and conection
  5. Kee the default printer name and location
  6. Print a test page :)
