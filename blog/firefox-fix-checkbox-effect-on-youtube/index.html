Fix Firefox Checkerbox Effect on Youtube

Eric Semeniuc

February 27, 2021

In Firefox on Linux, in fast moving scenes, you can see a checkerbox effect.

To fix this,

  1. Open about:config
  2. Change layers.acceleration.force-enabled to true
  3. Restart Firefox

That’s it!

Try this on a scene that has movement and high contrast, for example: https://youtu.be/0J6XKhn8LgY?t=27
