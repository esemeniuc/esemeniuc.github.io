Add Keyboard Shortcuts to Jupyter Lab

Eric Semeniuc

November 9, 2019

Via config file

Edit with your favourite editor:

~/.jupyter/lab/user-settings/@jupyterlab/shortcuts-extension/shortcuts.jupyterlab-settings

Via web interface

Settings -> Advanced Settings Editor -> Keyboard Shortcuts

Example to use Shift+D to restart and run all cells

{
    "shortcuts":[
        {
            "command":"runmenu:restart-and-run-all",
            "keys":[
                "Shift D"
            ],
            "selector":"[data-jp-code-runner]"
        }
    ]
}
